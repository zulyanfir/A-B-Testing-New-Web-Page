# Halaman Website Lama vs Baru, Mana yang Akan Meningkatkan *Conversion Rate*?

Sebuah halaman web baru telah dikembangkan dengan tujuan untuk meningkatkan jumlah pengguna yang melakukan pembelian produk perusahaan. Untuk mengevaluasi efektivitasnya, dilakukan perbandingan antara halaman lama dan halaman baru. Perusahaan berencana menerapkan halaman baru tersebut apabila terbukti memberikan dampak positif yang signifikan terhadap penjualan. Jika tingkat konversi dari halaman baru secara substansial lebih tinggi dibandingkan halaman lama, maka halaman tersebut akan diluncurkan secara resmi.
